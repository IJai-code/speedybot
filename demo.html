SpeedyBot

SpeedyBot is a Slack bot built with Node.js and Slack Bolt. It provides useful slash commands and API-powered responses.

Commands

/speedybot-ping
Checks bot latency.

/speedybot-help
Shows available commands.

/speedybot-catfact
Returns a random cat fact.

/speedybot-joke
Returns a random joke.

Technology

Built with:

  • Node.js
  • Slack Bolt
  • Axios
  • Slack API
  • Nest hosting

Deployment

SpeedyBot runs continuously on a Nest server using systemd, allowing it to stay online 24/7.
